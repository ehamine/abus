Detailed Description

A-Bus event RPC.

Usage: 

        cABusRequestEvent *my_rpc = my_abus->RequestEvent("examplesvc", "enter_pressed");

        my_rpc->append_str("typed_char", some_chars);
        my_rpc->publish();

        delete my_rpc;
